# Banking-Transaction-Fraud-Analysis
With the rapid growth of digital banking, UPI payments, online transfers, and card transactions, financial fraud has also increased significantly. Banks today face challenges in identifying suspicious activities such as phishing, card skimming, fake loan apps, and unauthorized transfers.

This project was built to understand how fraud detection works in real banking systems using SQL and structured data analysis.
